    Напишите программу, в которой циклом for значения переменной i будут меняться от 0 до 10 и посмотрите, как разные виды сравнений и операций работают в цикле.


  ```python
for i in range(10):
    print("i =", i)
    if i== 0:
        i += 2
    if i == 1:
        continue
    if i == 2 or i==3:
        print("переменная равна 2 или 3")
    elif i in [4,5,6]:
        print("переменная равна 4,5 или 6")
    else:
        break
```

## Краткий вывод:

Цикл for i in range(10): начинается и итерируется от 0 до 9 включительно.

На каждой итерации цикла программа выводит значение переменной i с помощью print("i =", i).

Первое условие if i == 0: проверяет, равно ли значение i нулю. Если i равно 0, то выполняется инструкция i += 2, что увеличивает значение i на 2. Важно отметить, что изменения значения i не влияют на текущий цикл, и цикл все равно перейдет к следующей итерации.

Далее идет условие if i == 1:, которое проверяет, равно ли значение i 1. Если i равно 1, то программа выполняет оператор continue, который прерывает текущую итерацию цикла и переходит к следующей итерации. В результате, код ниже этой точки на текущей итерации не будет выполняться.

Затем есть условие if i == 2 or i == 3:, которое проверяет, равно ли значение i 2 или 3. Если это условие выполняется, программа выводит сообщение "переменная равна 2 или 3".

Если ни одно из предыдущих условий не выполняется, программа переходит к условию elif i in [4, 5, 6]:, которое проверяет, присутствует ли значение i в списке [4, 5, 6]. Если i равно одному из этих значений, программа выводит сообщение "переменная равна 4, 5 или 6".

Если ни одно из условий выше не выполняется (то есть значение i не соответствует ни одному из условий), программа выполняет оператор break, который завершает выполнение цикла for досрочно. Таким образом, цикл завершается, и выполнение программы продолжается после цикла.
